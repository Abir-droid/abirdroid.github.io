How to updrade Windows Home to Pro for free?

As we know that we are the powerful windows user without any money. Even if we have some, we don't want to waste it on trival matters like windows upgrading. So today we are going to see how can we upgrade our Windows from Home edition to Pro.

For that first we have to know what is windows Home and Pro edititon. Windows Home editions are designed for home users. It includes features such as a new interface that is easier on the eyes and easier to use, wake on approach, lock on leave, smart app control, seamless redocking, live captions, natural narrator, Amazon Appstore, Windows Studio Effects, controller bar, auto HDR, widgets, sync your PC and phone, touchscreen support, and an improved search experience.
The **Pro** edition includes all the features of the Home edition and additional functionality focused on business and security³. It includes features such as the Group Policy Editor, Remote Desktop client, BitLocker encryption, Windows Update for Business, and more.
Here is the charted comparison of these two editions:

Windows Home Windows Pro
1. Designed for Home Users: Home editions are intended for individual consumers and home users who don't require advanced business features. 1. Business-Oriented Features: Pro editions are designed for small to medium-sized businesses and advanced users. They offer additional features and tools that are not available in the Home editions.
2. Simplified Interface: Home editions provide a straightforward and user-friendly interface, making them suitable for personal use. 2. Remote Desktop: Pro editions allow you to connect to your computer remotely using the Remote Desktop Protocol (RDP), which can be useful for remote administration and accessing your work computer from home.
3. Cortana: Cortana, Microsoft's digital assistant, is available in Home editions for tasks like voice search and assistance. 3. BitLocker: BitLocker Drive Encryption is available in Pro editions, allowing you to encrypt your entire hard drive to protect your data in case your device is lost or stolen.
4. Windows Update: Home editions receive automatic updates from Windows Update, helping users stay up to date with the latest security and feature updates. 4. Domain Join: Pro editions support domain joining, which is essential for businesses that use Active Directory domains for centralized management of users and devices.
5. Microsoft Store: You can access and download apps from the Microsoft Store in both Home and Pro editions, but Home editions may have certain restrictions in terms of app management for system administrators. 5. Group Policy: Pro editions provide more extensive Group Policy settings, allowing administrators to configure and control various aspects of the operating system.
6. No BitLocker: Home editions do not include BitLocker encryption, so if you need full-disk encryption, you'd need to use third-party software. 6. Hyper-V: Hyper-V is a virtualization platform that comes with Pro editions, enabling users to run virtual machines on their Windows systems.
7. No Remote Desktop Host: While you can use Remote Desktop to connect to other devices in Home editions, you cannot use your system as a Remote Desktop host (server).

So, from the comparision we can say that, windows Pro is the more featured edition than the Home. It has the all features from home including Hyper-V, Remote Desktops etc. So, lets see how can we upgrade to Pro for free.
For this we will be using cmd(windows command prompt). So 1st open the cmd window with adminstrator privileges.

Then Run these commands one by one:

dism /online /Get-TargetEditions
sc config LicenseManager start= auto & net start LicenseManager
sc config wuauserv start= auto & net start wuaserv
changepk.exe /productkey VK7JG-NPHTM-C97JM-9MPGT-3V66T

When input the last one and hit ENTER with for a second, the operator will start preparing the upgrade. Don’t turn off your PC. It will restart two times. Your Windows will upgrade successfully.

Note: This tutorial work for both windows 10 and windows 11.
